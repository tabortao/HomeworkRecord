# 📅 更新日志

## v1.0.7

- 添加任务表单的开始/结束日期功能。

## v1.0.6

- 移除强制底部显示样式，确保弹窗始终居中显示。优化输入框聚焦时的滚动行为，提升移动端输入体验。

## v1.0.5

- 优化移动端输入法适配和弹窗布局,添加任务弹窗输入法适配功能，防止输入框被遮挡,统一所有弹窗布局结构。

- 添加小程序图标；支持 PWA，可在移动设备上安装为应用程序。

- 增强任务管理安全性，添加、编辑、删除任务均需密码验证。

## v1.0.1

- 优化用户反馈功能，支持对任务添加评论。

## v1.0.0

- 初始版本，包含基本的任务管理、日历视图和统计功能。
